TotemUI - Come funziona?

Utilizzo

Se vedi questa finestra significa che la compilazione & l'esecuzione del programma è andata a buon fine, dunque
come funziona questo programma?

  1. Clicca in basso a sinistra per inserire un file
  2. Scegliere un file che finisce per .csv (esempio: data/Lista.csv)
  3. Una volta fatto il programma dovrebbe mostrare una lista alla tua sinistra
  4. Clicca uno degli elementi per visualizzare l'animale scelto
  5. Buon utilizzo del programma!

Compilazione

Per compilare il programma servono dei prerequisti

  • Il tuo compilatore dev'essere impostato per compilare wxwidgets
  • Il file eseguibile dev'essere eseguito dalla $(workspace) directory ovvero la stessa directory che contiene LICENSE & README.md
  • Bisogna compilare il file $(workspace)/test.cpp

Ovviamente il tuo ambiente di sviluppo può semplicemente compilare il file src/source.cpp a patto che sia impostato per creare un file eseguibile nella $(workspace) directory
